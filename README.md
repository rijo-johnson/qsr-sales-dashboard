## Dataset

**Pizza Place Sales** ‚Äî Maven Analytics  
A year of pizza sales data including:
- 21,350 orders
- 49,574 individual pizzas sold
- 32 unique pizza types across 4 categories (Classic, Veggie, Supreme, Chicken)
- Order timestamps spanning Jan 1, 2015 ‚Äì Dec 31, 2015

[Download from Maven Analytics Data Playground ‚Üí](https://mavenanalytics.io/data-playground)

To replicate this project:
1. Download the dataset (4 CSV files) from the link above
2. Open `pbix/QSR_Dashboard_v1.pbix` in Power BI Desktop
3. Update data source paths in Power Query if files are in a different location
